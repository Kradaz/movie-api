Overview

MyFlix is a REST API which stores a collection of top tier movies of all time into a database. It provides information about the movie such as: genre, author, and a small explanation of the movie. There are different types of movies from action to drama. The API uses quite a few CRUD operations to change data through HTTP and APi endpoints. It also allows users to register, delete or modify the data stored within an user account.
